### Step 1: Open PowerShell as Administrator

First, open PowerShell with **Run as Administrator**.

Paste the following command and press `Enter`:

```powershell

```

> **Note:** Make sure you have an active internet connection before running this command.

After running the command, a new window will open.

From the menu, **select Option 2**.
